# 📄 AI PDF Question Answering System (RAG + Streamlit)

This project is an **end-to-end Retrieval-Augmented Generation (RAG) system** that allows users to upload PDF files and ask questions about their content using an AI-powered chatbot.

It uses:
- LangChain for orchestration
- Hugging Face Transformers (FLAN-T5 model)
- FAISS for vector search
- Streamlit for UI

---

## 🚀 Features

- 📂 Upload multiple PDF files
- 🔍 Extract and process text automatically
- ✂️ Chunking for efficient retrieval
- 🧠 Semantic search using embeddings
- 💬 Conversational Q&A over PDFs
- ⚡ Fully local model (no paid API required)

---

## 🏗️ Tech Stack

- Python
- Streamlit
- LangChain
- Hugging Face Transformers
- Sentence Transformers
- FAISS (vector database)
- PyPDF2
